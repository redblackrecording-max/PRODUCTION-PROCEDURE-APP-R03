Quick Reference and Record Rules

• 1. Quick Reference

Border-clearing rule

On the CUSMA form, the HS Code name beside the description must match the HS Code in the next column.

1.1 HS Code Reference

HS Code

Name / Use

When to Use

• 8473.30.9100

Swap Module/Parts for Optical

Scanner

Small parts shipments, new scanners,

chassis/conveyor shipments, and

outbound swap/module or parts orders

• 8471.60.80.00

Optical Scanner & Recognition Device

New, rental, refresh, and full machine

swap shipments

• 9813.00.00

Canadian Goods Return

Full machine and module returns /

return items

1.2 Folder and Record Rules

•

Invoices are saved in the customer folder and are needed for OpenBorder booking.

•

OpenBorder Shipment Summary must be downloaded from the Production Email SHIPPING DOCS folder and saved to

the customer folder.

•

CUSMA, Canadian Customs, and Commercial Invoice documents must be stored with the shipment or return record.

•

QuoteID and shipping cost must be recorded in Commissioning Management.

•

Shipping cost must be entered in MRPeasy with vendor V00061 OpenBorder and item A00464 Shipping Cost.

• 2. End-to-End Workflows

2.1 Outbound Machine / Scanner / Module / Parts Shipment

• 1. Create customer order in MRP / MRPeasy.

• 2. Book stock and choose serialized lots when required.

• 3. Create invoice and select the correct HS Code.

• 4. Save invoice to the customer folder and print required copies.

• 5. Create CUSMA when required.

• 6. Book outbound shipment in OpenBorder.

• 7. Upload Commercial Invoice and CUSMA as required.

• 8. Record QuoteID, carrier details, tracking, and cost in Commissioning Management.

• 9. Create shipping-cost PO in MRPeasy.

• 10. Send HubSpot shipment notice or parts-shipped email.

2.2 Customer Return / RMA Shipment

• 1. Create the return/RMA record in MRP / MRPeasy.

• 2. Enter returned item, quantity, and RMA type.

• 3. Create invoice using HS Code 9813.00.00 and note Canadian Goods Return.

• 4. Complete Canadian Customs and CUSMA documents when required.

• 5. Book return in OpenBorder using customer address as pickup and TCG as delivery.

• 6. Upload Commercial Invoice, CUSMA, Canadian Customs Form, and Proof of Delivery.

• 7. Record QuoteID, carrier details, tracking, and cost in Commissioning Management.

• 8. Create shipping-cost PO in MRPeasy.

• 9. Send HubSpot return shipment notice and attach required files.

2.3 Swap Module / Parts Shipment

• 1. Create outbound item using the return-item style process in MRP / MRPeasy.

• 2. Create invoice using HS Code 8473.30.9100.

• 3. Set note to SWAP Module/Parts for Optical Scanner.

• 4. Save invoice to customer folder and print two copies: one for crate and one for driver.

• 5. Book shipment in OpenBorder and send HubSpot parts-shipped email.

MRP / MRPeasy Orders, Invoices, Returns, and Shipments

• 3. Procedure - MRP / MRPeasy Orders, Invoices, Returns, and Shipments

3.1 Create Customer Order

• 1. In Customer Orders, click Create.

• 2. Input customer company name.

• 3. Set Status to Confirmed.

• 4. Enter Account Number as the customer AN#.

• 5. Enter Serial Number as the TCGM#. This is usually the same as AN#, but not always; verify in the customer folder.

• 6. Under Product Type, enter the part number(s) of each item being sent to the customer.

• 7. Input Quantity.

• 8. Input Price per UoM. Use the Stock icon to find the cost under Cost.

• 9. Click Save.

• 10. Click Check stock and book items. Book all items; for items without stock, use Book Manually or create a Manufacturing

Order. For serialized items, book manually and select the correct lot.

3.2 Create Invoice

• 1. After items are booked, click the + under Invoices/Quotations.

• 2. Change Status to Unpaid.

• 3. Input Account Number and Serial Number.

• 4. Input the correct HS Code: 9813.00.00, 8471.60.80.00, or 8473.30.9100.

• 5. Under Free Text, include Country of Origin: Canada.

• 6. Click Save.

• 7. Click PDF at the top of the page to open the invoice.

• 8. Save the invoice into the customer folder for OpenBorder booking.

• 9. Print the required invoice copy/copies.

• 10. Use the I00 invoice number as the OpenBorder PO.

3.3 Create Shipment Record

• 1. Create a new shipment.

• 2. Enter delivery date from OpenBorder.

• 3. Enter tracking number.

• 4. Save.

• 5. Click Pick all items > Yes. If serialized, select the proper serial numbers being sent, then save.

3.4 Customer Returns / RMAs

• 1. Under CRM, create the return record for PhyzBatch-9000, module returns, refreshes, or swaps.

• 2. Input customer company name, return type, account number, serial number, and reason for return.

• 3. Input returned item: MA09 for PhyzBatch, MA01 for Scanner, KIT006 or KIT007 for Chassis/Conveyor swaps.

• 4. Enter part description in Free Text and input Quantity.

• 5. Add RMA Type as Receipt and Credit.

• 6. Create invoice, set Status to Unpaid, enter account and serial numbers, input HS Code 9813.00.00, and note Canadian

Goods Return.

• 7. Save invoice in the customer return folder and continue to the HubSpot procedure.

3.5 Outbound Swap Items

• 1. Input the outbound item using the same approach as Return Items.

• 2. Click Save.

• 3. Create Invoice and set Status to Unpaid.

• 4. Input Account Number and Serial Number.

• 5. Input HS Code 8473.30.9100.

• 6. Set note to SWAP Module/Parts for Optical Scanner.

• 7. Confirm product, quantity, and price are filled out.

• 8. Save, download invoice to customer folder for the outbound swap, and print two copies: one for crate and one for driver.

Creating a CUSMA Form

• 4. Procedure - Creating a CUSMA Form

• 1. Open CUSMA form in browser.

• 2. Enter part numbers in F. PART NUMBER / SKU. Each item is a new line.

• 3. Enter the description used in the Commercial Invoice under F. DESCRIPTION OF GOODS. Place HS Code names

beside the description.

• 4. Enter G. HS TARIFF CLASSIFICATION using the HS Code matching the HS Code name in the description column.

• 5. Use 8473.30.9100 for Swap Module/Parts for Optical Scanner.

• 6. Use 8471.60.80.00 for Optical Scanner & Recognition Device.

• 7. Use 9813.00.00 for Canadian Goods Return.

• 8. Set H. ORIGIN CRITERION to B.

• 9. Set I. COUNTRY OF ORIGIN to CA.

• 10. Export and download the document, then move it to the shipment folder.

• 11. Print required copies: one copy for shipments under $500 CAD, or two copies for machine shipments - one for the

shipping sleeve and one for the driver.

Creating a Canadian Customs Form

• 5. Procedure - Creating a Canadian Customs Form

• 1. Open Canadian Customs Form in browser.

• 2. Change Vendor to company name and address.

• 3. Fill out TAX ID from the customer information form in the customer folder.

• 4. Change the date of shipment.

• 5. Change other references to the correct Commercial Invoice number.

• 6. Line 11: Number of Packages - amount of packages being sent.

• 7. Line 12: Specification of commodities - description of items being returned.

• 8. Line 12 Quantity - repeat line 11.

• 9. Line 14: Unit price - price of items being sent, tax not included.

• 10. Line 15: Total - total price of all items being sent.

• 11. Line 16: Net / Gross - net weight and gross weight of items; gross will be the same as net.

• 12. Line 17: Invoice Total - carry down total from line 15.

• 13. Line 19: Exporter name and address - copy vendor information above except TAX ID.

• 14. Save the form, rename it with the company name at the end, and put it into the correct customer folder.

OpenBorder Outbound Shipment Booking

• 6. Procedure - OpenBorder Outbound Shipment Booking

6.1 Request Quote

• 1. Pick the date you are shipping the package or crate. FedEx will always do next-day pickup.

• 2. Set Pickup to our address.

• 3. Set Delivery to the customer address.

• 4. Set PO to the Commercial Invoice number.

• 5. Set Invoice Currency: small packages in CAD, crate shipments in USD.

• 6. Set Shipment Type: machine shipments are LTL; box shipments are Parcel.

• 7. Toggle Show Estimated Freight Class above Add Handling Unit.

• 8. Enter package information. Description is the parts being shipped or Card Sorting Machine / Scan Blaster Module.

• 9. Set Packing Type to Box, Crate, or Bag / Envelopes.

• 10. Enter Length, Width, and Height from the box/crate dimensions reference.

• 11. Weigh the package on the scale under the packing desk; packages under 1 lb default to 1 lb.

• 12. Leave Freight Class, NMFC Code, and Sub Code as defaults.

• 13. Add emergency contact as your name and use the work phone number +1 825 779 2636.

• 14. Set Value of Carriage to the cost of the package going out.

• 15. Set Currency to the Commercial Invoice currency (CAD or USD).

• 16. Do not insure crates when AirTags are added to each shipment.

• 17. Click Continue.

6.2 Carrier, Timing, Uploads, and Summary

• 1. For small packages, choose FedEx and no other carriers. Pick the cheapest option.

• 2. For crate shipments, pick the cheapest option if the shipment is not urgent.

• 3. Click Continue.

• 4. Check all customer information. Double-check address changes and confirm delivery start/end time.

• 5. Upload Commercial Invoice and CUSMA form from the customer folder.

• 6. Click Continue.

• 7. Copy QuoteID OB********** to Commissioning Management under Open Border Tracking and Courier Tracking. Add

carrier details.

• 8. Add total shipment price to OB Quote USD or OB Quote CAD depending on currency.

• 9. Click Finish Booking.

• 10. Check Production Email for SHIPPING DOCS, download Shipment Summary, and save it to the customer folder.

OpenBorder Return Shipment Booking

• 7. Procedure - OpenBorder Return Shipment Booking

• 1. Pick the date you are shipping the package or crate. FedEx will always do next-day pickup.

• 2. Set Pickup to the customer address.

• 3. Set Delivery to our address.

• 4. Set PO to the Commercial Invoice number.

• 5. Use CAD for small packages and USD for crate shipments.

• 6. Machine shipments are LTL; box shipments are Parcel.

• 7. Confirm whether the return is a box or crate with Customer Service. Incorrect packaging type can cause overweight fees.

• 8. Enter package information, dimensions, weight, contact, phone, value of carriage, and currency.

• 9. Choose FedEx for small packages and pick the cheapest option. For crates, pick cheapest unless urgent.

• 10. Double-check customer information, addresses, delivery time, and end time.

• 11. Upload Commercial Invoice, CUSMA form, Canadian Customs Form, and Proof of Delivery. Upload the Commercial

Invoice again for Proof of Delivery.

• 12. Copy QuoteID and carrier details into Commissioning Management.

• 13. Add total price to OB Quote USD or OB Quote CAD.

• 14. Click Finish Booking and save Shipment Summary from SHIPPING DOCS to the customer folder.

Creating the Shipping-Cost Purchase Order in MRPeasy

• 8. Procedure - Creating the Shipping-Cost Purchase Order in MRPeasy

• 1. Open MRPeasy in the browser and go to the Procurement app.

• 2. Create a Purchase Order.

• 3. Set Vendor to V00061 OpenBorder.

• 4. Change Currency to the currency used for the shipping cost: CAD or USD.

• 5. In Free Text, enter Account Number (AN#), Serial Number (TCGM#), and Customer Company/Name on separate lines.

• 6. Set Department to Shipping.

• 7. Set Type based on work being done: Shipping - Deployment, Shipping - Service & Part, or Shipping - Return.

• 8. Set Created By to your name.

• 9. Set Expected Date to approximate delivery date from the OpenBorder Booking Summary Form in the email inbox.

• 10. Set Order ID to the tracking number.

• 11. Set Shipped On to the shipment date.

• 12. Set Item to A00464 Shipping Cost.

• 13. Set Quantity to 1.

• 14. Input shipping cost under Price and ensure currency matches Commissioning Management.

• 15. Save.

HubSpot Shipping Communication

• 9. Procedure - HubSpot Shipping Communication

• 1. Open HubSpot in browser.

• 2. Find the CRM icon on the left side of the page.

• 3. Click Companies.

• 4. Search for the customer and click their name.

• 5. Create a new email.

• 6. Choose the main contact for the company. If unknown, find the main contact on page 2 of the customer information form

or under shipping contact.

• 7. Click Templates.

• 8. For full machine shipments, select 5 - SHIPMENT Notice. Paste the tracking URL from the Shipping tab in

Commissioning Management beside Tracking Number.

• 9. For returns, select SHIPPING - Return Shipment Notice. Fill in customer name, pickup via, day, month, date, and pickup

times. Upload return files using the paper clip icon.

• 10. Find packing lists at TCG Machines Documents > CRM and Support > Shipping > Packing Lists > choose correct one

for return.

• 11. For swap modules or part shipments, use the parts-shipped email template.

• 12. When all information is complete, send the email.

Parts / swap email wording

Hello _____,

Your Parts have been Shipped!

You can confirm this shipping information through the tracking number below.

Tracking Number: _________

If you have any difficulties or questions, please reach out and I will assist you.

Best regards,
